# VIGIL • Mark 13 — The Olivet Discourse

An interactive, beautifully illustrated web meditation on Mark 13 (ESV), built from the ESV Study Bible commentary.

## How to Use

1. Open `index.html` directly in any modern browser (Safari, Chrome, Firefox).
2. Use the **timeline** at the top to jump between the six movements.
3. Click any of the six illustrated cards to enter a rich **deep-dive meditation** containing:
   - Full Scripture for that section
   - Study Bible commentary
   - Key truths distilled from the text
   - A personal reflection prompt
   - A large, custom-generated illustration
4. Click **“MARK AS MEDITATED”** in any deep dive — your progress is saved automatically in the browser.
5. Use the top navigation to view the **full chapter text** at any time or reset your progress.

## Design Philosophy

- **Dark, reverent, vibrant**: Deep black/charcoal with selective gold, indigo, crimson, and teal accents.
- **Slow contemplation**: Every element encourages lingering rather than rushing.
- **Extensible**: The data model (sections array) is clean and ready for you to add Mark 1–12, 14–16, and beyond.

## Future Chapters

When you're ready to add the next chapter, create a new folder (e.g. `mark-14/`) and duplicate this structure. Or we can evolve this into a multi-chapter experience later.

## Assets

All seven illustrations were generated specifically for this meditation using a high-quality image model and are unique to this project.
